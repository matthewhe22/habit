# Pet yard background images

The pet yard renders these photos when a kid buys the matching background in
the shop. Add the four image files here with these **exact names**:

| File              | Background ("Scene Backgrounds" in shop) | Picture                          |
|-------------------|------------------------------------------|----------------------------------|
| `bg-backyard.jpg` | Backyard (🏡, 10 pts)                     | Sunny garden with wooden fence   |
| `bg-forest.jpg`   | Enchanted Forest (🌲, 20 pts)             | Green forest clearing            |
| `bg-fantasy.jpg`  | Fantasy Castle (🏰, 25 pts)               | Night castle over water          |
| `bg-beach.jpg`    | Sunny Beach (🏖️, 30 pts)                  | Beach with palm trees            |

Notes:
- Names and the `.jpg` extension must match exactly (they're referenced in
  `index.html` → `BACKGROUNDS`). Rename your files if needed.
- Landscape images look best (the yard is rendered ~300px tall, `cover`).
- If a file is missing, the yard falls back to the old gradient background, so
  nothing breaks.
